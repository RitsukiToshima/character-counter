# CLAUDE.md

このファイルは、Claude Code がこのプロジェクトを扱うときに参照する説明書です。

## プロジェクト概要ｃ

シンプルな文字数カウンターのWebアプリケーション。
入力されたテキストの文字数・行数を自動でカウントする。

## 技術スタック

- HTML / CSS / JavaScript（バニラ）
- ビルドツールなし
- `index.html` をブラウザで直接開いて動作確認する

## ファイル構成

- `index.html` — 画面のレイアウト
- `style.css` — 見た目のスタイル
- `script.js` — 動作のロジック
- `README.md` — 人間向けのドキュメント

## コーディング規約

- インデントは半角スペース2つ
- 文字列はダブルクォートで統一
- 関数名・変数名はキャメルケース

## 注意事項

- 外部ライブラリは追加しない（学習用のシンプル構成を維持）
- DOM操作は `script.js` に集約する
