# 文字数カウンター

シンプルな文字数カウンターのWebアプリケーションです。

## 機能

- 入力したテキストの文字数を自動でカウント
- スペースを含む文字数 / 除く文字数を表示
- 行数を表示
- クリアボタンで一括削除

## 使い方

`index.html` をブラウザで開くだけで動作します。

## ファイル構成

```
character-counter/
├── index.html   # 画面のレイアウト
├── style.css    # 見た目のスタイル
├── script.js    # 動作のロジック
├── CLAUDE.md    # Claude Code 向けの説明書
└── README.md    # このファイル
```
